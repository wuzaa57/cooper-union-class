These instructions assume that you have github installed, and the heroku toolbelt.

  1. checkout your github repo locally
  2. $term: chmod 600 ~/.ssh/*
  3. $term: heroku keys:add ~/.ssh/github_rsa.pub
  4. $term: heroku login
  5. $term: heroku create
  6. $term: git push heroku master
  7. $term: heroku open
